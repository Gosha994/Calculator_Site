# Calculator_Site
**Проект для получения информации о проекте Calculator Game**

Команда: 
- Лугов Святослав
- Семчик Максим


Описание проекта: Сайт с описанием проекта Calculator Game, возможностью его поддержки, обратной связи с разработчиками, 
а так же возможностью скачать Калькулятор. Этот сайт поможет узнать об основном проекте и зачем он нужен, что уже в нём реализовано и
какие перспективы на будущее у него есть.

Этапы разработки сайта:
- Создание фронтенда
- Создание бэкэнда
- Создание правильной базы данных
- Появление проекта на гитхабе
- Появление проекта на реплите
- Создание механизма обратной поддержки
- Создание механизма отзывов о проекте
- Создание механизма скачивания приложения
- Добавление дополнительных механик
