# Calculator_Site
**Проект для получения информации о разработках нашей команды**

**Команда сайта:** 
- Лугов Святослав
- Семчик Максим


**Описание проекта:** Сайт с описанием проекта Calculator Game, новостями о его разработке, возможностью его поддержки, обратной связи с разработчиками, 
а так же возможностью скачать Калькулятор. Этот сайт поможет узнать об основном проекте и зачем он нужен, что уже в нём реализовано и
какие перспективы на будущее у него есть.

**Этапы разработки сайта:**
- Создание бэкэнда
- Создание фронтенда
- Появление проекта на гитхабе
- Создание баз данных и добавление регистрации
- Появление озмоности скачивать наши разработки
- Добавления обратной поддержки и отзывов
- Добавление дополнительных механик
